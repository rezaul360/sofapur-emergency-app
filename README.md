# সফাপুর ইউ পি ইমার্জেন্সি সেবা অ্যাপ (Sofapur UP Emergency Service App)

সফাপুর ইউনিয়নের নাগরিকদের জন্য তৈরি একটি জরুরী যোগাযোগ ডিরেক্টরি এবং জনসেবামূলক
মোবাইল অ্যাপ্লিকেশন। এই অ্যাপ্লিকেশনের মাধ্যমে নাগরিকেরা খুব সহজেই চিকিৎসা,
পুলিশ, ফায়ার সার্ভিস, অ্যাম্বুলেন্সসহ বিভিন্ন প্রয়োজনীয় সেবার নম্বর খুঁজে পাবেন
এবং সরাসরি কল করতে পারবেন।

## 🚀 বৈশিষ্ট্যসমূহ (Features)

- **জরুরী কন্টাক্ট ডিরেক্টরি:** হাসপাতাল, থানা, ফায়ার সার্ভিস, অ্যাম্বুলেন্স এবং
  পল্লী বিদ্যুতের মতো গুরুত্বপূর্ণ সেবার ফোন নম্বরের তালিকা।
- **ওয়ান-ট্যাপ কলিং:** কোনো নম্বর কপি বা টাইপ না করে সরাসরি অ্যাপ থেকে এক
  ক্লিকেই কল করার সুবিধা।
- **রক্তের গ্রুপ ও ডোনার অনুসন্ধান:** রক্তের গ্রুপ অনুযায়ী ডোনারদের তালিকা এবং
  তাদের সাথে যোগাযোগের ব্যবস্থা।
- **অভিযোগ ও মতামত বক্স:** ইউনিয়ন পরিষদের সার্বিক উন্নয়নে নাগরিকেরা সরাসরি
  অ্যাপের মাধ্যমে হোয়াটসঅ্যাপে অভিযোগ বা মতামত জমা দিতে পারবেন।
- **গুরুত্বপূর্ণ সরকারি লিঙ্কসমূহ:** জন্ম-মৃত্যু নিবন্ধন, এনআইডি (NID) সেবা,
  ই-পাসপোর্ট আবেদন এবং ইউপি ট্যাক্স পোর্টালের মতো গুরুত্বপূর্ণ লিঙ্কসমূহ এক
  জায়গায়।
- **লগইন মডাল:** অ্যাপে প্রবেশের সময় ব্যবহারকারীর নাম দিয়ে স্বাগতম জানানোর
  চমৎকার ইউজার ইন্টারফেস।
- **অ্যানিমেটেড নোটিশ বোর্ড:** স্ক্রিনের ওপর দিয়ে চলমান নোটিশের মাধ্যমে ইউনিয়নের
  জরুরী ঘোষণা প্রচারের ব্যবস্থা।

## 🛠️ প্রযুক্তি (Tech Stack)

- **ফ্রেমওয়ার্ক:** React Native (Expo)
- **প্রোগ্রামিং ভাষা:** JavaScript / TypeScript
- **আইকন:** `@expo/vector-icons` (Ionicons)
- **ন্যাভিগেশন:** `@react-navigation/native` ও `@react-navigation/native-stack`
- **স্টাইলিং:** ক্লিন ও রিইউজেবল গ্লোবাল স্টাইলস স্ট্রাকচার (`globalStyles.js`)
